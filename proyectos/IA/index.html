En este espacio se presentan diferetnes códigos que he ehecho para solucionar problemas haceindo uso de herramientas de inteligenica artificial.

Tree.py

Este archivo es usado en muchos códigos, sirve para crear un objeto con propiedades para relacionarlos con otros nodos y sus valores.

tree.py

Puzzle.py

Este código busca organizar un rompecabezas de números (va de 1 a 4), requiere de conocer la solución del rompecabezas y el orden inical de los números, la salida del código es la secuencia de pasos para resolver el rompecabezas.

Puzzle.py

RoadTrip.py

El código busca crear la mejor ruta para ir de una ciudad a otra, para ello requiere las conexiones entre las carreteras (rutas) y se emplean las coordenadas para hacer un estimación de la distancia, el código retorna el mejor camino para ir de una ciudad a otra.

RoadTrip.py

GraphTraversal.py

Él codigo busca recorrer un camino de puntos, estos puntos conectan con otros puntos a determinada distancia, por lo que se busca es recorrer todos los puntos con el menor coste posible, el código retorna dicho camino.

GraphTraversal.py

Inequation.py

El código busca resolver una inecuación con dos incognitas, la idea es buscar el mayor valor posible para las variables.

7x + 4y ≤ 150
6x + 5y ≤ 160

inequation.py

SavingsAlgorithm

Este código busca crear rutas para entregar una serie de paquetes, la ruta debe ser la mejor, para ello se tiene en cuanta aquellos pedidos que se pueden conectar en una misma ruta y que no sobrepasen el peso máximo.

savingsAlgorithm.py

SearchMinCost

El código busca encontrar la mejor forma de organizar un pedido donde se tomen cuatro tipos y se compren uno por cada proveedor y aquel pedido tenga el menor coste.

searchMinCost.py
